# The Watch of Power
An Arduino based watch

## Required Parts
* [Arduino Uno](https://www.amazon.com/dp/B01EWOE0UU)
* [Waveshare 2.4 ince LCD Module](https://www.amazon.com/dp/B08H24H7KX)
* [AITRIP DS3231 Real Time Clock](https://www.amazon.com/dp/B09KPC8JZQ)
